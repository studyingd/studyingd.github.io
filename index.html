# 介绍

# ☕Java全栈

Java全栈是指掌握多种技能，能够独立开发应用程序的前端和后端以及完成各项数据库工作。其中前端通常是指用户在应用程序中能看到或交互的部分，后端是应用程序中处理业务逻辑，数据库交互，用户认证，服务端配置等部分。全栈开发需要掌握至少一门编程语言，PHP、Java、C、Python等，运用开发框架和第三库，前端技术，数据库与缓存，基本的设计能力。

# 🌍计算机网络

计算机网络是指将地理 (opens new window)位置不同的具有独立功能的多台计算机 (opens new window)及其外部设备，通过通信线路连接起来，在网络操作系统 (opens new window)，网络管理软件 (opens new window)及网络通信协议 (opens new window)的管理和协调下，实现资源共享 (opens new window)和信息传递的计算机系统。

计算机网络 (opens new window)主要是由一些通用的、可编程的硬件互连而成的，而这些硬件并非专门用来实现某一特定目的（例如，传送数据或视频信号）。这些可编程的硬件能够用来传送多种不同类型的数据，并能支持广泛的和日益增长的应用。

# 🛰️前端开发

前端开发是创建WEB (opens new window)页面或APP (opens new window)等前端界面呈现给用户的过程，通过HTML (opens new window)，CSS (opens new window)及JavaScript (opens new window)以及衍生出来的各种技术、框架、解决方案，来实现互联网产品的用户界面 (opens new window)交互。

前端开发从网页制作 (opens new window)演变而来，名称上有很明显的时代特征。在互联网 (opens new window)的演化进程中，网页制作是Web1.0时代的产物，早期网站主要内容都是静态，以图片和文字为主，用户使用网站的行为也以浏览为主。随着互联网技术 (opens new window)的发展和HTML5、CSS3的应用，现代网页更加美观，交互效果显著，功能更加强大。

前端开发 (opens new window)跟随移动互联网 (opens new window)发展带来了大量高性能的移动终端设备 (opens new window)应用。HTML5 (opens new window)，Node.js (opens new window)的广泛应用，各类UI (opens new window)框架，JS类库 (opens new window)层出不穷，开发难度也在逐步提升。
